## Shared Structure

- Fixed header/navigation appears on all routes.
- Footer appears on all routes.
- Most non-home pages use a cyan page banner followed by a light grey content body.
- News/post pages use the same local post data and card component.
- Contact/news/about pages include the shared support CTA band.

## Home

1. Fixed translucent header over hero.
2. Full-width image hero with centered headline.
3. Purple/cyan split CTA strip.
4. Awards-night gallery.
5. Latest News three-card section.
6. Testimonial cyan band.
7. Join Us image CTA.
8. Shared footer.

Home contact/social widget is intentionally removed per baseline requirement.

## Interaction Models

- Header: click-driven mobile drawer, fixed desktop nav.
- Forms: visual-only baseline, no submit behavior.
- News cards: link-driven.
- Past Winners: static, year-grouped data.
- FAQ: static displayed answers matching current visible state.
