# MVF Technical Stack Analysis

## Current Site

- Platform: Wix/Thunderbolt.
- Content: Wix static pages, Wix blog widgets, Wix forms.
- Styling: generated Wix CSS with Avenir-like font assets, purple/cyan/navy palette, fixed header.
- Assets: Wix Static Media CDN.

## Clone Implementation

- Framework: Next.js 16 App Router.
- React: 19.
- Styling: Tailwind CSS v4 tokens in `globals.css`.
- Components: plain typed React components with reusable data-driven sections.
- Assets: local files in `public/images/mvf/`.
- Forms: visual-only baseline by requirement.
- Donation: support page placeholder pending separate Stripe integration.
