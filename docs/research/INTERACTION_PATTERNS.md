# MVF Interaction Patterns

- Header navigation: direct link navigation with mobile drawer open/close.
- Donate: routes to `/support-us` until Stripe work lands.
- Contact form: visual only; fields are present for layout but no network submission.
- Newsletter form: visual only; button has no submit action.
- Application forms: external `.doc` links preserved from current site.
- Blog posts: internal links via `/post/[slug]`.
- FAQ: currently displayed as open/static answers rather than hidden accordion panels.
